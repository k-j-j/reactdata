r_basic
 "homepage": "https://k-j-j.github.io/curr",
,
    "predeploy": "npm run build",
    "deploy": "gh-pages -d build"


해당 프로젝트내에서
npm install -save gh-pages
안되면
https://github.com/k-j-j/basic_rt

npm install gh-pages --save-dev

package.json 파일에
"homepage": "http://깃주소",

package.json의 scripts에 predeploy와 deploy 요소를 추가

아래의 빨간 부분을 추가하면 된다.

"scripts": {
    "start": "react-scripts start",
    "build": "react-scripts build",
    "test": "react-scripts test",
    "eject": "react-scripts eject",
    "predeploy": "npm run build",    // build script를 호출하여 build 디렉터리 생성
    "deploy": "gh-pages -d build"    // gh-pages를 호출하고 build 디렉터리를 업로드
}

리액트 프로젝트를 deploy하기 위해 터미널에서 아래 명령을 실행

npm run deploy
 

그러면 build 디렉터리가 만들어졌을 것이다.
 build 폴더에서 바로 git init 부터실행하여 업로드 ==> settings => pages 작업
